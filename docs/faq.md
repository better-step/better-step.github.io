# Frequently Asked Questions

**Q1: What is the purpose of Better Step?**
A: Converts proprietary STEP files into an open HDF5 format, enabling data access without expensive licenses.

**Q2: How do I install Better Step?**
A: Refer to the [Installation](installation.md) page. Install with pip or from source.

**Q3: Which Python versions are supported?**
A: Python 3.7 and later.

**Q4: I encountered an error during installation. What should I do?**
A: Review the Installation page or open a GitHub issue with error details.

**Q5: How can I contribute?**
A: See the [Contributing](contributing.md) page for instructions and guidelines.
